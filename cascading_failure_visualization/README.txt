%%Sameet Sreenivasan, 2014

Movie of a simulated cascading failure on data representing the European power grid.

Individual (png) frames were generated using Gephi, and combined into a movie using ffmpeg using the command line tool; see here: https://trac.ffmpeg.org/wiki/Create%20a%20video%20slideshow%20from%20images

Captions for these movies can be found in the following publication under Supporting Information:
http://www.plosone.org/article/info%3Adoi%2F10.1371%2Fjournal.pone.0084563
